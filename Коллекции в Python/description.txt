Задача
Цели домашнего задания
Проверить умение работать с коллекциями в Python: списками, кортежами, множествами, словарями.
Научиться генерировать объекты с помощью list, set, dict comprehensions.
Проверить умение работать с функциями round, zip.


Что входит в домашнее задание
1. Обязательные задачи

Comprehensions и операции над множествами.
Дежурство.
Расстояние между двумя точками.
Учёт товаров.


2. Дополнительные задачи

Частота слова — часть 1.
Частота слова — часть 2.
Площадь треугольника.

Критерии оценки
программа выдаёт верный ответ на разных наборах входных данных;
код читабелен: переменным даны осмысленные названия, отступы и правила расстановки пробелов соблюдены;
код декомпозирован с помощью функций.
